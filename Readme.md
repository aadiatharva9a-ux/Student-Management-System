# Student Record Management System Using Python

## Project Description

The Student Record Management System is a Python-based application designed to manage student data efficiently. It allows users to add, view, search, update, and delete student records. The system stores data in a JSON file, ensuring that records are saved permanently.

## Problem Statement

Managing student records manually can lead to errors, data loss, and difficulty in searching or updating information. This project provides a simple digital solution to manage student records efficiently.

## Features

* Add student record
* View student records
* Search student by roll number
* Update student details
* Delete student record
* Save data using file handling

## Technologies Used

* Python
* JSON file handling
* Functions
* Lists and Dictionaries

## How to Run the Project

Step 1: Install Python

Step 2: Download the project files

Step 3: Open terminal or command prompt

Step 4: Run the program
