Student Record Management System (Python)
About This Project

This project is a simple Student Record Management System made using Python. I created this program to practice file handling and basic Python concepts. The system helps in managing student details in a digital way instead of writing everything on paper.

With this program, a user can add new student records, see all records, search for a specific student, update information, and delete records if needed. The data is saved in a JSON file, so the information is not lost when the program is closed.

This project is mainly designed for learning purposes and can be useful for beginners who want to understand how real-world problems can be solved using Python.

What This Program Can Do
Add a new student record
Display all student records
Search for a student using roll number
Update student information
Delete a student record
Save data permanently using a file
Tools and Technologies Used
Python
JSON file
Functions
Lists and Dictionaries
Visual Studio Code (for writing the code)
How the Program Works

The program shows a menu on the screen. The user selects an option by typing a number. Based on the choice, the program performs the required action like adding, searching, or deleting a student record.

All student data is stored in a file called:

students.json

This file keeps the data safe even after the program is closed.

How to Run This Project

Follow these simple steps:

Install Python on your computer
Download or clone this project
Open the project folder
Open terminal or command prompt
Run the program using this command:
python main.py

After running the program, you will see the menu and can start using the system.

Example Menu
1. Add Student
2. View Students
3. Search Student
4. Update Student
5. Delete Student
6. Exit

Just enter the number of the option you want to use.
